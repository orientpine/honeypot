# 제3장 사업 목표 및 추진 전략

## 1. 최종 연구 목표

### 최종 연구 목표

> **다수·다종의 건설기계가 사람의 개입 없이 동적 환경에 적응하며 자율적으로 협력 작업을 수행하는 AI 건설기계 다개체 협업플랫폼 개발**

| 디지털 트윈 | 자율작업 | 인공지능 |
|:---:|:---:|:---:|
| 실시간 디지털 트윈 / 다개체 산업기계 안전 운영 | 다개체 센서 융합 인지 / 다중 협동 제어 | 숙련자 데이터 기반 AI 모델 / 숙련자-AI 자율작업기계 |

#### 디지털트윈 기반 다개체 운영 및 관제 기술
- 실시간 다개체 디지털트윈 모델링 및 운영 최적화 기술 개발
- 고정밀 센서 연동 기반 다개체 상태 모니터링 및 관제 기술 개발

#### 인공지능 기반 다개체 협력 환경 인식 기술
- 다중센서 융합 및 AI 기반 위치·상태 인식 기술 개발
- 복합 환경에서 협력 작업을 위한 건설 현장 3차원 인식 기술 개발

#### 숙련자 데이터 기반 다개체 협업 계획 기술
- 숙련자 작업 패턴 학습 기반 협업 계획 자동화 기술 개발
- 다개체 협업 최적 경로 및 작업 분담 계획 수립 기술 개발

#### 숙련자 데이터 기반 자율작업 제어 기술
- 숙련자 조작 데이터 기반 자율작업 제어 알고리즘 개발
- 비상 상황 대응을 위한 자율 복구 제어 기술 개발

#### Zero 사고 산업기계 다개체 협업 자율화 실증
- 다개체 협업 자율작업 안전성 검증 및 실증 기술 개발
- 사고 예방형 자율 운전 및 작업 통합 실증 기술 개발
